Russian Phonics Learning App

区域,逻辑,UI 元素
Header,字母大/小写平铺,字符需使用内置 IPA 字体渲染
名称音区,展示字母的称呼音,名称音标 (IPA) + 播放按钮
拼读音区,区分硬/软辅音（动态渲染）,硬音 IPA + 按钮 / 软音 IPA + 按钮
示例单词区,2-3 个代表性单词,单词（带重音符号）+ 单词 IPA + 翻译 + 按钮

数据字典
字段名,类型,说明
letter_id,Integer,字母唯一标识 (1-33)
name_audio,String/Blob,字母名音频文件路径
phonetic_audio,String/Blob,基础拼读发音路径
is_consonant,Boolean,是否为辅音 (用于判断是否显示硬/软对比)
example_list,Array,关联的示例单词 ID 列表

元音规则
规则分类,规则描述,IPA 变化,示例词 1 (短),示例词 2 (长)
O 的弱化 (1级),O 在重音节前一个音节读作 [a],o→[a],окно́ [ak′no],молоко́ [malak′o]
O 的弱化 (2级),O 在其他非重音节读更短的 [ə],o→[ə],о́блако [ˈobləkə],хорошо́ [xərɐˈʂo]
E/Я 的弱化,E/Я 在非重音节读作 [i],e/я→[i],язы́к [i′zɨk],девя́тый [dji′vjatɨj]

辅音规则
规则分类,规则描述,IPA 变化,示例词 1 (短),示例词 2 (长)
词尾清化,浊辅音位于单词末尾时，读对应的清辅音,"b→[p],d→[t] 等",сад [sat],клуб [klup]
辅音丛清化,浊辅音在清辅音前时，变为清辅音,v+t→[ft] 等,вто́рник [ˈftornjɪk],авто́бус [ɐfˈtobʊs]
辅音丛浊化,清辅音在浊辅音前时，变为浊辅音,s+b→[zb] 等,футбо́л [fʊˈdbol],вокза́л [vɐɡˈzal]

特殊符号
规则分类,规则描述,功能,示例词 1,示例词 2
软音符号 (ь),使前面的辅音变软，不发音,辅音 → 软辅音,мать [matj],слова́рь [slɐˈvarj]
硬音符号 (ъ),起分音作用，使后面元音带 [j],隔断发音,объе́кт [ɐbˈjekt],подъе́зд [pɐˈdjest]
